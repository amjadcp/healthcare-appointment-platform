Appointment Management

Endpoints for booking, slots, auditing, and RabbitMQ DLQ management

Reprocess message(s) from DLQ (Admin only)

Republishes a failed event (by eventId) back to the main exchange, or reprocesses all messages in the DLQ if no ID is specified.

Authorizations:
Bearer Authentication
query Parameters
eventId
string

Optional eventId of a specific message to reprocess

Responses

Generated server url

http://localhost:8080/api/v1/events/dlq/reprocess

Dismiss/Purge message(s) from DLQ (Admin only)

Discards a specific message from the DLQ by eventId, or purges all DLQ messages if no ID is specified.

Authorizations:
Bearer Authentication
query Parameters
eventId
string

Optional eventId of a specific message to dismiss

Responses

Generated server url

http://localhost:8080/api/v1/events/dlq/dismiss
